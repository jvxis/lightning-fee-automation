Bem-vindo ao LND Fee Automation

Automação de Taxas para Node Lightning

Esta aplicação permite automatizar as taxas do seu node Lightning Network (LND) com base no fluxo de entrada e saída de cada canal e nas taxas dos peers.

Selecione uma das opções abaixo para começar:

Dashboard

Visualize o status do seu node e as estatísticas de taxas em tempo real.

Acessar Dashboard
Canais

Gerencie seus canais e ajuste as taxas individualmente.

Gerenciar Canais
Configurações

Configure as estratégias e parâmetros da automação de taxas.

Configurar
Recursos Principais
  • Ajuste automático de taxas com base no fluxo dos canais
  • Monitoramento das taxas dos peers para ajustes competitivos
  • Múltiplas estratégias de ajuste de taxas
  • Interface web intuitiva para visualização e configuração
  • Histórico de taxas e desempenho dos canais
  • Controle granular sobre quais canais automatizar
